Hand Gesture Following Bot

Visit GitHub

Tech stack: Python, Raspberry Pi, Arduino, OpenCV, PyGame, GPIO, IR Sensors, TensorFlow

Description: The Hand Gesture Following Bot is an innovative project that combines Python programming with IoT technology to detect and respond to human hand gestures in real-time. The bot uses infrared (IR) sensors and motors to detect movement and perform tasks based on user gestures. For instance, a raised hand could command the bot to stop, while a swiping gesture could control its movement. The bot was developed using Raspberry Pi as the central processing unit, and Python libraries like OpenCV for gesture detection, as well as GPIO for controlling the motors. This project demonstrates an intuitive human-robot interaction model, paving the way for gesture-controlled automation in various industries.

Image preprocessing techniques, such as normalization and resizing, are applied to prepare the images for training and inference.
